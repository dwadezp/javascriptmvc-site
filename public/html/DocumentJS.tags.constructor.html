DocumentJS.tags.constructor  class     

Source

Documents a constructor function and its parameters.

Example:

/*
 * @class Customer 
 * @constructor
 * Creates a new customer.
 * @param {String} name
 */
 var Customer = function(name) {
    this.name = name;
 }
