    can.proxy  function     

    Source

    can.proxy(function) accepts a function and returns a new one that will always the context from which it was called. This works similar to jQuery.proxy.

     var func = can.proxy(function(one){
   return this.a + one
 }, {a: "b"}); 
 func("two") //-> "btwo"
