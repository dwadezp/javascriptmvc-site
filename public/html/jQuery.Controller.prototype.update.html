jQuery.Controller.prototype.update  function     

Called if an controller's jQuery helper is called on an element that already has a controller instance of the same type. The base method extends this.options with the options passed in. If you overwrite this, you might want to call this._super.

Example

$.Controller("Thing",{
  init: function( el, options ) {
    alert( 'init:'+this.options.prop )
  },
  update: function( options ) {
    this._super(options);
    alert('update:'+this.options.prop)
  }
});
$('#myel').thing({prop : 'val1'}); // alerts init:val1
$('#myel').thing({prop : 'val2'}); // alerts update:val2
